dndchar
=======

DnD Character Sheet

This was built from a yeoman generator and barely works. None of the grunt stuff seems very useful, but whatever.

Dependencies
======
* Node (0.10.x)
* Ruby


Setup
=======
1. clone repo
2. ```npm install```
3. ```bower install```
4. ```bundle install```
5. ```grunt server```
